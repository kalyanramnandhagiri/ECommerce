# E-Commerce Web Application

## 🚀 Tech Stack
- Java
- Spring Boot
- React.js
- MySQL
- REST APIs

## 💡 Features
- User Authentication
- Product Management
- Cart & Checkout
- Razorpay Payment Integration
- Order Management

## 📌 Description
Developed a full-stack E-commerce web application with secure payment integration using Razorpay. Implemented RESTful APIs and responsive UI for seamless user experience.
